# 🌌 Supernova AI

Supernova AI is an AI-powered career guidance platform built using Google Gemini 2.5 Flash. It helps students discover personalized career paths based on their education, interests, skills, and aspirations.

## Features

- AI-powered career recommendations
- Career Match Score
- Personalized learning roadmap
- Skill development suggestions
- Certification recommendations

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- Google Gemini 2.5 Flash API

## How to Run

```bash
npm install
npm start
```

Create a `.env` file:

```
GEMINI_API_KEY=YOUR_API_KEY
```

Then open:

```
http://localhost:3000
```
